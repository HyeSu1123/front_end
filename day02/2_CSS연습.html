CSS는 HTML 문서의 스타일을 지정하는데 사용하는 언어이다. CSS는 HTML 요소가 이렇게 표시되어야 하는지 설명한다.

CSS란?

  • CSS는 Cascading Style Sheets 의 약자이다.
  • CSS는 한번에 여러 웹페이지(HTML파일)에 적용됙 수 있다.
  • 확장자 CSS는 별도로 외부파일에 CSS를 작성할 때 사용한다.
  • CSS 작성은 style 태그를 사용한다. 특정 요소에 적용할선택자(Selector)가 필요하다.
  • CSS 프로퍼티는 글꼴의 크기, 색상, 배경 이미지, 테두리 등등이 있다. 모든 태그에 공통적인 것 또는 특정 태그에만 적용되는 것을 구별해서 사용한다.
